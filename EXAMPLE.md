# PyMD Editor 使用示例

欢迎使用 **PyMD Editor**！这是一个功能演示文档。

## 基本格式

### 文本样式

- **粗体文本**
- *斜体文本*
- ~~删除线~~
- `行内代码`

### 列表

有序列表：
1. 第一项
2. 第二项
3. 第三项

无序列表：
- 项目 A
- 项目 B
  - 子项目 B.1
  - 子项目 B.2

任务列表：
- [x] 已完成的任务
- [ ] 待办任务
- [ ] 另一个待办事项

## 代码块

```python
def hello_world():
```

## 试试快捷键！

- `Ctrl+N` - 新建文档
- `Ctrl+O` - 打开文件
- `Ctrl+S` - 保存
- `Ctrl+T` - 切换深色/浅色主题
- `Ctrl+Shift+W` - 导出为 Word
- `Ctrl+Shift+P` - 导出为 PDF
