## 🎯 How to Insert Images

### Method 1: Use the Insert Image Feature
1. Press `Ctrl+Shift+I` 
2. Or click "File → Insert Image"
3. Or use the 🖼️ button in the AI panel

### Method 2: Drag and Drop
1. Drag image files directly into the editor
2. The Markdown syntax will be automatically generated

### Method 3: Manual Markdown
```markdown
![Alt Text](image-path.png)
![Description](https://example.com/image.jpg)
```

## 🔧 Image Features
- **Auto-sizing**: Images are responsive (max-width: 100%)
- **Styling**: Rounded corners and subtle shadows
- **Path Resolution**: Supports local files, URLs, and relative paths
- **Format Support**: PNG, JPG, GIF, BMP, SVG
