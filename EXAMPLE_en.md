# PyMD Editor Example

Welcome to **PyMD Editor**! This is a feature demo document.

## Basic Formatting

### Text Styles

- **Bold text**
- *Italic text*
- ~~Strikethrough~~
- `Inline code`

### Lists

Ordered list:
1. First item
2. Second item
3. Third item

Unordered list:
- Item A
- Item B
  - Subitem B.1
  - Subitem B.2

Task list:
- [x] Completed task
- [ ] Todo task
- [ ] Another todo

## Code Blocks

```python
def hello_world():
    print("Hello from PyMD Editor!")
    return True
```

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
}
```

## Try Shortcuts!

- `Ctrl+N` - New document
- `Ctrl+O` - Open file
- `Ctrl+S` - Save
- `Ctrl+T` - Toggle theme
- `Ctrl+Shift+W` - Export Word
- `Ctrl+Shift+P` - Export PDF
