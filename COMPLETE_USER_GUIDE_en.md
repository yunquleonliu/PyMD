## ❓ Q1: How to associate .md files with the editor?

### ✅ Best answer: Drag & Drop (zero setup)

**Steps:**
```
1. Right-click run_editor.bat → Send to desktop shortcut
2. Drag any .md file onto the shortcut icon
3. Editor opens the file automatically
```

**Why recommended:**
- ✅ Zero setup (no config needed)
- ✅ Zero permissions (no admin needed)
- ✅ Zero learning (just drag & drop)
- ✅ Zero risk (doesn't modify system)

**Other methods:**
- Method 2: Right-click → Open with (one-time setup)
- Method 3: Registry association (needs admin)

📖 See: `FILE_ASSOCIATION_GUIDE_en.md`
