# 🖼️ PyMD Editor 图片插入使用指南

## 📋 快速使用方法

### 方法一：使用插入图片功能（推荐）
1. **快捷键**：按 `Ctrl+Shift+I`
2. **菜单**：点击"文件" → "插入图片"  
3. **AI面板**：点击右侧AI助手中的"🖼️ Insert Image"按钮
4. 在弹出的文件对话框中选择图片文件
5. 系统会自动生成正确的Markdown语法并插入到编辑器

### 方法二：拖放操作
1. 直接将图片文件从文件管理器拖拽到编辑器中
2. 系统自动生成Markdown图片语法

### 方法三：手动输入
```markdown
![图片描述](图片路径)
```

## 🎯 图片路径说明

### 在线图片（完全支持）
```markdown
![在线图片](https://example.com/image.jpg)
```
✅ **推荐使用**：在线图片显示最稳定

### 本地图片路径建议
由于WebEngine安全限制，本地图片可能无法直接显示。**建议解决方案**：

1. **使用相对路径**：将图片放在文档同一目录下
   ```markdown
   ![本地图片](./my-image.png)
   ```

2. **创建images文件夹**：在文档目录建立专门的图片文件夹
   ```markdown
   ![示例图片](./images/example.jpg)
   ```

3. **使用在线图床**：推荐上传到图床服务
   - GitHub仓库
   - 图床服务网站
   - 云存储服务

## ⚠️ 重要提示

**本地图片限制**：由于浏览器安全策略，本地文件路径可能在预览中无法显示，但导出PDF/Word时会正常包含。**最佳实践是使用在线图片URL或将文档和图片放在同一目录下使用相对路径。**

## 💡 最佳实践

1. **在线图片**：适合临时预览和分享
2. **相对路径**：适合本地项目文档
3. **图片规格**：支持PNG、JPG、GIF、BMP、SVG格式
4. **自动调整**：图片会自动适应预览窗口宽度
