# AI功能使用指南

## 🤖 PyMD Editor AI助手使用说明

### 快速开始

1. **打开应用**: 运行 `python -m src.pymd_editor.main`
2. **选择模式**: 选择"三栏布局模式"标签页
3. **AI面板**: 右侧面板为AI助手
4. **开始使用**: 在编辑器中输入文本，点击AI按钮

### ✨ AI功能按钮

| 按钮 | 功能 | 使用场景 |
|------|------|----------|
| **✨ Improve Text** | 文本优化 | 润色文章、改进表达、提升专业性 |
| **📝 Summarize** | 内容总结 | 提取要点、生成摘要、快速理解 |
| **🌐 Translate** | 翻译 | 中英互译、多语言转换 |
| **💡 Brainstorm** | 头脑风暴 | 创意生成、方案建议、灵感启发 |

### 🎯 使用技巧

#### 1. 文本选择
- **选中文本**: AI只处理选中的部分
- **无选择**: AI处理整个文档内容
- **建议**: 选择段落或章节进行精准优化

#### 2. 最佳实践
- **文本优化**: 选择需要润色的段落
- **内容总结**: 选择长篇文章或章节
- **翻译功能**: 选择具体的句子或段落
- **创意生成**: 输入主题关键词

#### 3. 响应查看
- AI回复显示在右侧面板下方
- 支持滚动查看长回复
- 状态栏显示处理进度

### 🔄 工作流程示例

#### 文档写作流程
1. **草稿编写** → 在左侧编辑器写初稿
2. **AI优化** → 选择段落，点击"✨ Improve Text"
3. **查看建议** → 在右侧查看AI优化建议
4. **应用修改** → 手动采纳AI建议修改文档
5. **预览效果** → 中间面板实时预览

#### 内容研究流程
1. **资料收集** → 粘贴研究资料到编辑器
2. **智能总结** → 点击"📝 Summarize"获取要点
3. **创意拓展** → 基于总结点击"💡 Brainstorm"
4. **翻译需求** → 使用"🌐 Translate"处理外文

### ⚙️ 个性化设置

#### 布局调整
- **AI密集模式**: 扩大右侧AI面板
- **专注写作**: 隐藏AI面板专心写作  
- **预览优先**: 主要查看渲染效果
- **编辑器独占**: 全屏编辑模式

#### 快捷操作
- 使用布局下拉菜单快速切换
- 锁定/解锁分割器调整
- 拖拽分割线自定义宽度

### 🌐 AI服务说明

#### Personal AI (默认免费)
- **完全免费**: 无需API密钥
- **中文优化**: 支持中文提示和回复
- **快速响应**: 1-2秒获得结果
- **多功能**: 支持所有AI任务类型

#### Gemini API (可选配置)
- **高质量**: Google最新AI模型
- **需要配置**: 提供API密钥
- **按量计费**: 根据使用量收费
- **智能切换**: 免费服务不可用时自动使用

### 🚨 注意事项

1. **网络连接**: AI功能需要互联网连接
2. **内容隐私**: 文本会发送到AI服务处理
3. **处理时间**: 复杂任务可能需要几秒钟
4. **结果参考**: AI建议仅供参考，请人工审核

### 🔧 故障排除

#### 常见问题

**Q: 点击按钮没有反应？**
A: 检查网络连接，确保编辑器中有内容

**Q: 显示"Error"状态？**  
A: 可能是网络问题或API服务暂时不可用

**Q: AI回复显示不完整？**
A: 在响应区域滚动查看完整回复

**Q: 想要更高质量的AI回复？**
A: 配置Gemini API密钥以使用高级AI服务

#### 技术支持
- 查看控制台输出了解详细错误
- 确认`httpx`依赖已正确安装
- 检查防火墙是否阻止网络请求
