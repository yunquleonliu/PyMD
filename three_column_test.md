# Three Column Layout Test

Welcome to the **Three Column Layout** mode! 

## Features

The three-column layout provides:

### Left Panel: Editor
- Traditional Markdown editing
- Syntax highlighting
- Auto-completion

### Middle Panel: Preview  
- Real-time HTML preview
- Custom CSS themes
- Responsive rendering

### Right Panel: AI Assistant
- Quick actions
- Content suggestions
- Smart improvements

## Layout Options

You can switch between different layout presets:

- **Default** (40-40-20): Balanced layout for general use
- **Focus Writing** (60-40-0): Hide AI panel, focus on writing
- **AI Intensive** (30-30-40): Expand AI panel for heavy assistance
- **Preview Only** (40-60-0): Emphasize preview for final review
- **Editor Only** (100-0-0): Full-screen editing mode

## Quick Actions

Try these AI-powered quick actions:
- ✨ **Improve Text**: Enhance selected text
- 📝 **Summarize**: Create concise summaries  
- 🌐 **Translate**: Multi-language translation
- 💡 **Brainstorm**: Generate creative ideas
