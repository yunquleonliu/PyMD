# PyMD Editor - WYSIWYG功能使用指南

## 概述

PyMD Editor现在支持两种编辑模式：

1. **传统模式**：左边是Markdown源码编辑器，右边是实时预览
2. **所见即所得模式**：可以直接在渲染后的内容上进行编辑

## WYSIWYG功能特性

### ✅ 已实现的功能

1. **模式切换**
   - 点击"编辑模式"按钮进入WYSIWYG编辑
   - 点击"预览模式"按钮切换回只读预览
   - 两个模式之间内容自动同步

2. **基本编辑**
   - 直接在预览窗格中编辑文本
   - 支持复制、粘贴操作
   - 实时内容变化检测

3. **格式化工具栏**
   - **B** 按钮：应用粗体格式
   - **I** 按钮：应用斜体格式  
   - **H** 按钮：转换为标题
   - **Link** 按钮：创建链接

4. **智能转换**
   - HTML内容自动转换为Markdown格式
   - 支持多种HTML元素的识别和转换
   - 保持格式的准确性

### 🔄 使用方法

1. **启动应用程序**
   ```bash
   cd "c:\Users\Leon Liu\Desktop\PyMD"
   python -m src.pymd_editor.main
   ```

2. **切换到WYSIWYG模式**
   - 点击界面底部的"所见即所得"标签页
   - 或者直接在该标签页中开始编辑

3. **开始编辑**
   - 点击"编辑模式"按钮
   - 直接在渲染的内容上点击和编辑
   - 使用工具栏按钮进行格式化

4. **保存内容**
   - 内容会自动同步到传统编辑器
   - 使用Ctrl+S保存文件

### 💡 使用建议

1. **对于简单编辑**：使用WYSIWYG模式更直观
2. **对于复杂格式**：使用传统模式更精确
3. **混合使用**：可以在两种模式间自由切换

### 🐛 已知限制

1. 复杂表格的编辑可能不够精确
2. 某些特殊Markdown语法可能需要在传统模式下编辑
3. 大文档可能存在性能影响
